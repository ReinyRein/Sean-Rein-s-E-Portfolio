        <h2>Quarter 1 ICT 9 Lessons</h2>
        <p>During the first quarter of ICT 9, students are introduced to the basics of CSS, which is used to style HTML elements and improve the visual presentation of web pages.</p>
        
        <h3>CSS Syntax</h3>
        <p>CSS (Cascading Style Sheets) uses selectors and properties to apply styles to HTML elements. The syntax follows a `selector { property: value; }` format. For example, `p { color: blue; }` changes the text color of all paragraph elements to blue.</p>
        
        <h3>Selectors</h3>
        <p>Selectors are used to target specific HTML elements. Common selectors include element selectors (`p`, `h1`), class selectors (`.classname`), and ID selectors (`#idname`). Advanced selectors allow styling based on attributes, pseudo-classes, and pseudo-elements.</p>
        
        <h3>Text and Font Styling</h3>
        <p>CSS enables customization of text appearance using properties like `font-family`, `font-size`, `color`, `text-align`, and `text-decoration`. This helps enhance readability and design consistency.</p>
        
        <h3>Box Model</h3>
        <p>The box model consists of four areas: content, padding, border, and margin. Understanding this model is crucial for spacing and layout adjustments. For example, `padding` adds space inside the element, while `margin` creates space outside.</p>
        
        <h3>Backgrounds and Colors</h3>
        <p>CSS allows setting background colors, images, and gradients using properties like `background-color`, `background-image`, and `background-gradient`. This enhances the aesthetic appeal of web pages.</p>
        
        <h3>Layouts</h3>
        <p>Layouts can be structured using techniques like Flexbox and Grid, which provide efficient ways to align and distribute elements on a page. These methods are essential for creating responsive designs.</p>
