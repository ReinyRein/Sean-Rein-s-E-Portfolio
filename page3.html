Quarter 3 ICT 9 Lessons

During the first quarter of ICT 9, students are introduced to the basics of JavaScript, a powerful programming language used to create interactive and dynamic web pages.

What is JavaScript?

JavaScript is a scripting language that enables developers to implement complex features on web pages, such as interactive forms, animations, and real-time updates.

JavaScript Syntax

JavaScript code is written in a structured way, using variables, functions, and event listeners. A simple example of JavaScript syntax is:

            
                let message = "Hello, World!";
                console.log(message);

Variables and Data Types

JavaScript uses variables to store data. Common data types include strings, numbers, and booleans. Example:

            
                let name = "John";
                let age = 25;
                let isStudent = true;

Functions

Functions allow reusable blocks of code. Example:

            
                function greet(name) {
                    return "Hello, " + name + "!";
                }
                console.log(greet("Alice"));

Event Handling

JavaScript can respond to user interactions. Example of a button click event:

            
                document.getElementById("myButton").addEventListener("click", function() {
                    alert("Button Clicked!");
                });

Reflection

The process of learning JavaScript via ICT 9 has been revolutionary. Initially, it appeared as if comprehending the interaction among JavaScript, HTML, and CSS was complex; however, with each step forward with the lessons on variables, functions, and event handling, it was apparent how powerful and indispensable JavaScript is when it comes to web development. One of the most exciting things was how to make a webpage interactive, whether through simple alerts, interactive buttons, or user input handling. The event-handling tutorial was the most interesting because it described how JavaScript handles user input, turning webpages into interactive pages. Despite the challenges that JavaScript's syntax and debugging mechanism pose, the experimentation and practice with code have actually enhanced my problem-solving skills. The first course this quarter has provided a good foundation, and I hope to have the opportunity to explore more advanced areas such as DOM manipulation and APIs in the future.
