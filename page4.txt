        <h2>Quarter 1 ICT 9 Lessons</h2>
        <p>This lesson focuses on advanced JavaScript concepts including Events, Pop-Up Boxes, Conditions, Controls, and Loops.</p>
        
        <h3>Events</h3>
        <p>JavaScript events are actions that occur on a webpage, such as clicking a button or submitting a form. Event listeners are used to respond to these actions.</p>
        
        <h3>Pop-Up Boxes</h3>
        <p>JavaScript provides alert, confirm, and prompt pop-ups to interact with users. These allow notifications, confirmations, and input retrieval.</p>
        
        <h3>Conditions</h3>
        <p>Conditional statements like if-else allow JavaScript to execute different code blocks based on conditions. Example:</p>
        <pre>
            <code>
                if (age >= 18) {
                    console.log("You are an adult.");
                } else {
                    console.log("You are a minor.");
                }
            </code>
        </pre>
        
        <h3>Controls & Loops</h3>
        <p>Loops such as for and while allow repetition of code blocks until conditions are met. Example:</p>
        <pre>
            <code>
                for (let i = 0; i < 5; i++) {
                    console.log("Iteration: " + i);
                }
            </code>
        </pre>
